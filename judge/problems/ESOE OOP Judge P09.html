Description

Write a program that can serve as a simple calculator. This calculator keeps track of a single number (of type double double ) that is called result result and that starts out as 0.00 0.00 . Each cycle allows the user to repeatedly add, subtract, multiply, or divide by a second number. The result of one of these operations becomes the new value of result result . The calculation ends when the user enters the letter R R for “result” (either in upper- or lowercase).

A command is composed of “operator” “operator” and “value” “value” separated by a space. There are five five kinds of unknown command exceptions: First, the operator is any operator symbol other than + + , - - , * * , or / / . Second, the value can’t be parsed as a real number. Third, the operator and the value are both invalid. Fourth, the command is not formed by exact one one operator and one one value separated by one one space. Last, the command indicates dividing by zero zero . An UnknownCmdException UnknownCmdException will be thrown if one of these exceptions described above happens. You need to pass the error message to UnknownCmdException UnknownCmdException class in order to figure out what happens. The system will call exception.getMessage() exception.getMessage() to check it out.

The error message format is as below:
1. 1. invalid operator: " [operator] is an unknown operator [operator] is an unknown operator "
2. 2. invalid value: " [value] is an unknown value [value] is an unknown value "
3. 3. both invalid: " [operator] is an unknown operator and [value] is an unknown value [operator] is an unknown operator and [value] is an unknown value "
4. 4. invalid form: " Please enter 1 operator and 1 value separated by 1 space Please enter 1 operator and 1 value separated by 1 space "
5. 5. dividing by zero: " Can not divide by 0 Can not divide by 0 "

1. public void calResult(String cmd) throws UnknownCmdException public void calResult(String cmd) throws UnknownCmdException – parse the command and set new result result . If command is not valid, throws an UnknownCmdException UnknownCmdException contained the error message.

2. public String getMsg() public String getMsg() – output the message of the result result of this step. The output message depends on the calculation count count . The output format is as below:
(1)when starting the calculation, it will return " Calculator is on. Result = [result] Calculator is on. Result = [result] "

3. public boolean endCalc(String cmd) public boolean endCalc(String cmd) – determine whether calculation comes to the end or not.

String[] cmd_arr = cmd_str.split(",");
for (int i = 0; i < cmd_arr.length; i++) {
  try {
    if (cal.endCalc(cmd_arr[i])) 
      break;		
    cal.calResult(cmd_arr[i]);
    System.out.println(cal.getMsg());
  } catch (UnknownCmdException e) {
    System.out.println(e.getMessage());
  }
}
