Pizza

Deadline

2019-06-24 23:59:59.

Description

Create a class named Pizza Pizza that stores information about a single pizza. It should contain the following:

1. 1. Private instance variables to store the size of the pizza(either small, medium or large), the number of cheese toppings, the number of pepperoni toppings, and the number of ham toppings. String for size and int for the others.
2. 2. Default constructor that set size to small, the other three instance variables to 1.
3. 3. Constructor that can set all of the instance variables.
4. 4. Public methods to get and set the instance variables.
5. 5. A public method named calcCost() calcCost() that return a double double that is the cost of the pizza. Pizza cost is determined by:

Small: $10 + $2 per topping
Medium: $12 + $2 per topping
Large: $14 + $2 per topping
For example, a large pizza with one cheese, one pepperoni and two ham toppings should cost a total of $22.

6. 6. An overload method equals() equals() that determine whether this pizza is the same as the other one by their size, the number of cheese toppings, the number of pepperoni toppings, and the number of ham toppings.
7. 7. An override method toString() toString() that return a String containing the pizza size, quantity of each topping. The output format is as below:
size = small, numOfCheese = 0, numOfPepperoni = 0, numOfHam = 0

p2.getSize() p2.getSize() , return a string.
p2.getNumberOfCheese() p2.getNumberOfCheese() , return an integer which is number of cheese of the pizza.
