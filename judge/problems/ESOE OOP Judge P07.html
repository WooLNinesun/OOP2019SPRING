Description

When saving objects, array is a good choice, but sometimes it is inconvenient due to its fixed size. Therefore, you have to implement a new data structure – a resizable-array called SimpleArrayList SimpleArrayList class to save data. The class uses an Integer Integer array as its internal instance variable to save integer, and its size needs to be changed automatically based on user's manipulation. As elements are added to a SimpleArrayList SimpleArrayList , its size grows automatically. On the contrary, its size decreases while elements are removed from the array. To make this class useful, implement it with the following methods:

1. Default constructor that sets the array size to zero.
2. Constructor that can set initial array size, and set all Integer Integer to zero in the array.
3. public void add(Integer i) public void add(Integer i) – appends the specified element to the end of this array.
4. public Integer get(int index) public Integer get(int index) – returns the element at the specified position in this array. If the specified position is out of range of the array, returns null null .
5. public Integer set(int index, Integer element) public Integer set(int index, Integer element) – replaces the element at the specified position in this array with the specified element, and returns the original element at that specified position. If the specified position is out of range of the array, returns null null .
6. public boolean remove(int index) public boolean remove(int index) – If a null null element is at the specified position, returns false false ; otherwise removes it and returns true true . Shifts any subsequent elements to the left if removes succesfully.
7. public void clear() public void clear() – removes all of the elements from this array.
8. public int size() public int size() – returns the number of elements in this array.
9. public boolean retainAll(SimpleArrayList l) public boolean retainAll(SimpleArrayList l) – retains only the elements in this array that are contained in the specified SimpleArrayList SimpleArrayList . In other words, removes from this array all of its elements that are not contained in the specified SimpleArrayList SimpleArrayList . Returns true true if this array changed as a result. See examples below:

Example 9-1: Example 9-1:
list = {1, 2, 3, 1, 2, 3, 1} list = {1, 2, 3, 1, 2, 3, 1} , having elements 1 1 , 2 2 , and 3 3 .
list2 = {5, 5, 4, 4, 3, 3, 2, 2, 1, 1} list2 = {5, 5, 4, 4, 3, 3, 2, 2, 1, 1} , having elements 1 1 , 2 2 , 3 3 , 4 4 , and 5 5 .
Because list2 list2 has all the elements which list list has, after list.retainAll(list2) list.retainAll(list2) is executed, list list remains the same:
list = {1, 2, 3, 1, 2, 3, 1} list = {1, 2, 3, 1, 2, 3, 1}
Since there is not any change in list list , list.retainAll(list2) list.retainAll(list2) returns false false .

Example 9-2: Example 9-2:
list = {1, 2, 3, 1, 2, 3, 1} list = {1, 2, 3, 1, 2, 3, 1} , having elements 1 1 , 2 2 , and 3 3 .
list2 = {2, 2, 1, 1, 0, 0} list2 = {2, 2, 1, 1, 0, 0} , having elements 0 0 , 1 1 , and 2 2 .
Because only 1 1 and 2 2 are common elements between list list and list2 list2 , after list.retainAll(list2) list.retainAll(list2) is executed, only 1 1 and 2 2 remains in list list , which means the element 3 3 is removed from list list :
list = {1, 2, 1, 2, 1} list = {1, 2, 1, 2, 1}
Since there are changes in list list , list.retainAll(list2) list.retainAll(list2) returns true true .

list = new SimpleArrayList(initialSize); list = new SimpleArrayList(initialSize); , where initialSize initialSize is an non-negative integer.
list.add(i) list.add(i) , where i i is an Integer Integer object.
list.get(index) list.get(index) , where index index is an integer indicating the position.
list.set(index, element) list.set(index, element) , where index index is an integer indicating the position, and element element is an Integer Integer object.
list.remove(index) list.remove(index) , where index index is an integer indicating the position.
list.retainAll(l) list.retainAll(l) , where l l is a SimpleArrayList SimpleArrayList object.

list.set(index, element) list.set(index, element) returns an Integer Integer object, which is the element at the index index position before this setting.
list.remove(index) list.remove(index) returns true true if it removes successfully, returns false false otherwise.
list.size() list.size() returns an integer showing the number of elements in this array.
list.retainAll(l) list.retainAll(l) returns true true if list list is changed, returns false false otherwise.
