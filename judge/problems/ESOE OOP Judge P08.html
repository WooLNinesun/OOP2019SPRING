Description

Given an abstract class named Shape Shape with some methods and variables that can get information of this geometric shape, please define Triangle Triangle (treat it as regular triangle), Circle Circle and Square Square classes that extend Shape Shape class and implement all abstract methods. For Circle Circle class, length length means its diameter.

For extensibility and maintenance, you should create a factory class named ShapeFactory ShapeFactory to create all kinds of Shape Shape for main class. It contains an enum enum named Type Type that store all kinds of shape. It also contains a method named createShape createShape to create instance of all shapes refer to Type Type parameter.

Please set the mathematical constants π π with the aid of Math Math class.

The following test code outputs the area and perimeter of the Triangle Triangle , Square Square and Circle Circle . The output result should round off to the 2nd decimal place round off to the 2nd decimal place .

(You can add your own instance variables or methods in any class if you want.)

Input Format

In Shape.java Shape.java :

protected double length protected double length , stores the side length side length for regular triangle, square, and stores the diameter diameter for circle.
public abstract void setLength(double length) public abstract void setLength(double length) , set the length of this shape.

In ShapeFactory.java ShapeFactory.java :

enum Type enum Type , contains 3 kinds of shapes as values.
public Shape createShape(ShapeFactory.Type shapeType, double length) public Shape createShape(ShapeFactory.Type shapeType, double length) , where shapeType shapeType can refer to Triangle, Square or Circle while length length represents the side length or diameter of the shape.

Output Format

In Shape.java Shape.java :

public abstract double getArea() public abstract double getArea() , get the area of this shape.
public abstract double getPerimeter() public abstract double getPerimeter() , get the perimeter of this shape.
public String getInfo() public String getInfo() , print the area and perimeter of this shape. The output format is as below: " Area = 47.74, Perimeter = 31.5 Area = 47.74, Perimeter = 31.5 "
If the number is zero, the output should be " 0.0 0.0 ".

In ShapeFactory.java ShapeFactory.java :

public Shape createShape(ShapeFactory.Type shapeType, double length) public Shape createShape(ShapeFactory.Type shapeType, double length) , returns an object which 'is-a' Shape object.

Sample Input

ShapeFactory shapeFactory = new ShapeFactory();
Shape triangle = shapeFactory.createShape(ShapeFactory.Type.Triangle, 7.5);
Shape square = shapeFactory.createShape(ShapeFactory.Type.Square, 5);
Shape circle = shapeFactory.createShape(ShapeFactory.Type.Circle, 5);
System.out.println(triangle.getInfo());
System.out.println(square.getInfo());
System.out.println(circle.getInfo());
System.out.println(square.getArea() > triangle.getArea());
System.out.println(square.getPerimeter() > circle.getPerimeter());
triangle.setLength(10.5);
square.setLength(3.2);
circle.setLength(0);
System.out.println(triangle.getInfo());
System.out.println(square.getInfo());
System.out.println(circle.getInfo());
System.out.println(square.getArea() > triangle.getArea());
System.out.println(square.getPerimeter() > circle.getPerimeter());
